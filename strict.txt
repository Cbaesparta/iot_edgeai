
Boot Jetson Orin and connect keyboard, mouse, monitor.

Connect Jetson to internet temporarily for package install (Ethernet or WiFi).

Open Terminal on Jetson.

Update system:
sudo apt update
sudo apt upgrade -y

Install required system packages:
sudo apt install -y git python3-pip python3-opencv mosquitto mosquitto-clients network-manager

Go to home folder:
cd ~

Clone your project (if already on GitHub):
git clone <your-repo-url> jetson-ai-iot

If not using GitHub, copy project folder to Jetson manually and then go into it:
cd ~/jetson-ai-iot

Install Python dependencies:
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt

Make scripts executable:
chmod +x scripts/install_autostart.sh
chmod +x scripts/wait_for_camera.sh
chmod +x scripts/setup_hotspot.sh
chmod +x scripts/disable_hotspot.sh

Create runtime environment file:
cp .env.example .env

Edit environment values if needed:
nano .env
Change only if required:
CAMERA_INDEX=0
FLASK_HOST=0.0.0.0
FLASK_PORT=5000
MQTT_HOST=127.0.0.1
MODEL_NAME=yolov8n.pt

Save and exit nano:
Ctrl+O, Enter, Ctrl+X

Install and enable auto-start services (no login needed):
sudo bash scripts/install_autostart.sh

Check backend service:
systemctl status jetson-ai-backend.service

Check dashboard service:
systemctl status jetson-ai-dashboard.service

Check MQTT broker:
systemctl status mosquitto

Find Jetson IP on your network:
hostname -I

From phone/laptop browser on same WiFi router, open:
http://JETSON_IP:8501
http://JETSON_IP:5000/video
http://JETSON_IP:5000/api/status
http://JETSON_IP:5000/api/detections
http://JETSON_IP:5000/api/alerts

If service is not running, restart:
sudo systemctl restart jetson-ai-backend.service
sudo systemctl restart jetson-ai-dashboard.service

To view logs live:
journalctl -u jetson-ai-backend.service -f
journalctl -u jetson-ai-dashboard.service -f

Reboot test (important):
sudo reboot

After reboot, verify auto-start again:
systemctl is-active jetson-ai-backend.service
systemctl is-active jetson-ai-dashboard.service

Plug/unplug camera test:
Unplug camera for 5 to 10 seconds, plug back in, then check stream URL again.

Optional: make Jetson itself a WiFi hotspot (phone/laptop connect directly):
sudo bash scripts/setup_hotspot.sh JetsonCamNet Jetson1234 wlan0

Connect phone/laptop to:
SSID: JetsonCamNet
Password: Jetson1234

Open via hotspot IP:
http://10.42.0.1:8501
http://10.42.0.1:5000/video

If hotspot IP differs, find it:
ip -4 addr show wlan0

Optional: disable hotspot auto-connect:
sudo bash scripts/disable_hotspot.sh

You are fully deployed when all three are true:
jetson-ai-backend.service is active
jetson-ai-dashboard.service is active
Stream opens on phone and laptop
